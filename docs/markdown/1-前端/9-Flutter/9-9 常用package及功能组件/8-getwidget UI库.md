# getwidget UI库


`GetWidget` 是一个开源的 `Flutter UI` 库，提供了一系列漂亮、响应式且可定制的 `UI` 组件，用于快速构建 `Flutter` 应用程序。该库的目标是简化开发流程，提高开发效率，同时提供现代化和吸引人的用户界面。
