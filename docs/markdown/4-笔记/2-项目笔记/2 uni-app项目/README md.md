## 项目安装和运行

1. 克隆本项目到本地

```bash
git clone [https://github.com/yourusername/yourproject.git](https://github.com/yourusername/yourproject.git)
```

1. 安装依赖

```bash
npm install
```

1. 运行项目

```bash
npm run dev
```

1. 打包项目

```bash
npm run build
```
